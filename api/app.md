# Points App
## Описание приложения
Приложение, которое используется для отправки точек на координатную плоскость с последующей проверкой факта попадания/непопадания.

## Описание частей приложения
Приложение состоит из клиентской (веб-приложение на React) и серверной (Spring Boot) частей. Взаимодействие между back-end и front-end обеспечивается посредством REST API.

## Описание функциональности приложения
1. Функциональность, связанная с пользователем
   - Добавление пользователя
   - Подтверждение аккаунта пользователя
   - Поддержка двухэтапной аутентификации
   - Возможность восстановления аккаунта пользователя
   - Возможность удаления аккаунта пользователя
2. Функциональность, связанная с попаданиями (для всех пользователей)
   - Получение коллекции попаданий конкретного пользователя
   - Добавление попадания конкретным пользователем
   - Удаление попадания конкретным пользователем
3. Функциональность, связанная с областями (для пользователя с ролью админа)
   - Изменение областей, с которыми будут работать пользователи веб-приложения
