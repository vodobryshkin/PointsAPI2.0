# Points App API 2.0
## Стек
- Java 21
- Spring Boot 4
- RabbitMQ (очередь сообщений на отправку электронной почты)
- Redis (ссылки на подтверждение аккаунта)

## Описание функциональности
1. Функциональность, связанная с пользователем
    - Регистрация нового пользователя
    - Подтверждение аккаунта пользователя
    - Выдача пользователю токена для аутентификации
    - Поддержка двухэтапной аутентификации с использованием One Time Password
    - Выдача пользователю прав и их проверка
2. Функциональность, связанная с попаданиями (для всех пользователей)
    - Получение коллекции попаданий конкретного пользователя
    - Добавление попадания конкретным пользователем
    - Получение количества попаданий конкретного пользователя
3. Функциональность, связанная с областями (для пользователя с ролью админа)
    - Изменение областей, с которыми будут работать пользователи веб-приложения
4. Функциональность, связанная с электронной почтой:
    - Реализация отправки сообщений через очередь сообщений при следующих событиях:
      - Подтверждение аккаунта после регистрации. Письмо должно содержать ссылку, при переходе по которой аккаунт подтвердится
      - После корректной обработки имени пользователя и пароля при входе в аккаунт. Письмо должно содержать 6-значный OTP-код для подтверждения входа в аккаунт
