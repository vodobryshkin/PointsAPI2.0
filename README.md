# Points App API 2.0
## Описание проекта
Учебный проект, направленный на знакомство с Spring Framework и написанием различной backend-функциональности используя различные его компоненты.
Это разработанный бэкенд приложения на базе Spring Boot, которое осуществляет проверку попадания точки в заданную область на координатной плоскости. Помимо этого, реализована система пользовательских аккаунтов и возможность админу менять области, которые будут отображаться для всех пользователей.

Проект представляет собой обновлённую версию API для работы с попаданиями на координатную плоскость для [лабораторной работы №4](https://github.com/vodobryshkin/lab4-backend) по дисциплине "Веб-Программирование".

## Запуск
1. Склонировать к себе репозиторий
2. В корне проекта создать .env-файл с переменными окружения, которые нужны в properties
3. Прописать в терминале
    ```shell
    ./mvnw clean package
    docker build -t points-api2-app .
    docker run -d --rm --network host --env-file .env points-api2-app
    ```
## Роли
Реализовано три роли: UNVERIFIED_USER, USER, ADMIN. У каждой роли свои возможности:

| Технические различия                                                     | UNVERIFIED_USER | USER | ADMIN |
|--------------------------------------------------------------------------|-----------------|------|-------|
| Запрашивать письмо электронной почты с ссылкой на подтверждение аккаунта | +               | -    | -     |
| Вход в приложение (логин+пароль+OTP)                                     | +               | +    | +     |
| Отправка точки на проверку на попадание                                  | -               | +    | +     |
| Получение списка попаданий для пользователя                              | -               | +    | +     |
| Получение количества попаданий для пользователя                          | -               | +    | +     |
| Получение списка текущих областей на координатной плоскости              | -               | +    | +     |
| Изменение областей, которые будут видны во всем приложении               | -               | -    | +     |
